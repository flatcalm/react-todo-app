State
useState
지역 변수의 값을 바꿔줄 수 있는 JS의 클로저와 같은 기능.
useState 함수는 여러 개도 사용 가능



useState와 같은 기능을 하는 바닐라 JS의 클로저 함수
전역 변수 -> 장점 값을 유지할 수 있음.
	단점 값을 참조할 수 있어 변경될 수 있다.
지역 변수 -> 장점 외부에서 참조할 수 없어 값을 변경할 수 없음.
	단점 값을 유지할 수 없음.

- 지역 변수의 장점을 살리고 단점을 보완하기 위해 클로저 함수를 사용.
- 함수 안에서 함수를 리턴하는 방식으로 작성.
